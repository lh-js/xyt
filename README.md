# lh-xyt

#### 介绍
萌新用uniapp+unicloud做的一个简单的校园服务平台（只测试了微信小程序端）

#### 软件架构
uniapp+unicloud全家桶


#### 安装教程

1.  首先要在unicould上创建一个服务空间（推荐使用阿里云）
2.  右键unicloud目录关联一个创建好的服务空间
3.  右键cloudfunctions目录上传所有云函数，右键database目录上传所有db schema和校验函数
4.  手动在web控制台创建emailCode，idle-class-one，idle-class-two，idle-class-three表
5.  启动
